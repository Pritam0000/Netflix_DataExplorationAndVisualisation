# Netflix Content Analysis Dashboard

This Streamlit application provides an interactive dashboard for analyzing Netflix content data. Users can explore various aspects of the Netflix catalog, including content type distribution, top producing countries, rating distribution, genre popularity, and more.

## Features

- Interactive filters for content type and release year
- Visualizations for content overview, including distribution charts and trends
- Duration analysis for movies and TV shows
- Word cloud generation from content descriptions
- Data explorer for detailed examination of the dataset

## Installation

1. Clone this repository:
   ```
   git clone https://github.com/your-username/netflix-analysis.git
   cd netflix-analysis
   ```

2. Install the required packages:
   ```
   pip install -r requirements.txt
   ```

3. Run the Streamlit app:
   ```
   streamlit run app.py
   ```

## Data

This application uses the Netflix dataset (`netflix.csv`). Make sure to place this file in the project root directory before running the app.
